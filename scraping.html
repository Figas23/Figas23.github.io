EMODnet project

As a member of the EMODnet project I perform Web Scraping to gather river data (such as flow, temperature, ...) from both URLs and APIs using libraries such as Requests, BeautifulSoup and Selenium. The project is in operation in a AWS EC2 machine as we retrive this daily data from over 700 stations with the cron job task scheduler and made it available an FTP server and in the ERDDAP software. Since there are many stations, I monitor which ones are returning data via a map where they stations are marked with the colors red (haven't retrieve in the last 30 days), yellow (haven't retrieve in the last 7 days) or green.

You can check part of my personal experience in web scraping from this GitHub repository:Web Scraping Repository.
