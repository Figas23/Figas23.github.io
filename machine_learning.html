Satellite Derived Bathimetry

I have professional and personal experience developing and deploying Machine Learning models either from Scikit-Learn as well as creating my own Convolutional and Recurrent Neural Networks. I also worked with Machine Learning for time series analysis in order to forecast Electricity Prices using algorithms such as ARIMA and SARIMA, Facebook Prophet and Long Short Term Memory (LSTM) networks which are Neural Networks, suitable to work with time dependencies.

Currently I have been working on a Machine Learning project to predict bathimetry (sea depth) with Satellite imagery (using different wavelenghts of the sea surface reflectance). This approach seeks to sidestepping the high expenses associated with traditional methods as it provides a more accessible and efficient means of creating detailed depth maps for oceans, lakes, and rivers.

I applied cross-validated ML models from Scikit-Learn such as RandomForest and XGBoost, evaluating the models using metrics like MAE, RMSE and R2-score and performing hyperparameter tuning with GridSearchCV and RandomSearchCV.

I was able to obtain very satisfactory results where prediction had an RMSE of about 0.7 meters for a group of more than 10000 data points.
